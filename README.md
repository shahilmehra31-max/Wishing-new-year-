# Love New Year Project

Ye ek romantic New Year website hai jo long-distance relationship ke liye design ki gayi hai.  

## Files
- `index.html` → Main webpage
- `style.css` → Styling aur animations
- `script.js` → Next button aur page navigation
- `letter1.jpg` → Handwritten letter 1
- `letter2.jpg` → Handwritten letter 2

## Instructions
1. Sab files ek hi folder me rakho.
2. `index.html` browser me open karo (Chrome / Firefox recommended)
3. Next button click karke pages flip karo.
4. Images aur background music automatically load ho jaayenge.
